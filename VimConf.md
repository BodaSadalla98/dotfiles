Vim shortcuts
=============

General shortcuts
----------------
F2 --> Save file

F3 --> Toggle the spell check






NERDTree
-------

F4 --> Toggle the NERDTree menu

Taglist
------

crtlp
--------
Crtl + p --> open the search menu

Esc --> closes the menu
